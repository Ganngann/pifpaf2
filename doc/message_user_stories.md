# User Stories pour la Messagerie

Ce document contient les user stories pour les fonctionnalités de messagerie.

## Fonctionnalités de base (Déjà implémentées)

*   **US-MSG-001 : Démarrer une conversation**
    *   En tant que acheteur potentiel, je veux pouvoir démarrer une conversation avec un vendeur à partir de la page d'un article, afin de poser des questions sur l'article.

*   **US-MSG-002 : Envoyer un message**
    *   En tant que utilisateur, je veux pouvoir envoyer un message dans une conversation, afin de communiquer avec l'autre partie.

*   **US-MSG-003 : Voir les messages**
    *   En tant que utilisateur, je veux pouvoir voir tous les messages d'une conversation, afin de suivre l'historique de la discussion.

*   **US-MSG-004 : Voir mes conversations**
    *   En tant que utilisateur, je veux pouvoir voir une liste de toutes mes conversations, afin d'accéder facilement à mes discussions.

## Améliorations

### Notifications

*   **US-MSG-005 : Notification de nouveau message**
    *   En tant que utilisateur, je veux recevoir une notification lorsque je reçois un nouveau message, afin d'être informé rapidement et de pouvoir répondre.

*   **US-MSG-006 : Compteur de messages non lus**
    *   En tant que utilisateur, je veux voir un compteur de messages non lus sur l'icône de messagerie, afin de savoir combien de messages je n'ai pas encore lus.

### Gestion des conversations

*   **US-MSG-007 : Archiver une conversation**
    *   En tant que utilisateur, je veux pouvoir archiver une conversation, afin de nettoyer ma boîte de réception sans supprimer la conversation.

*   **US--MSG-008 : Supprimer une conversation**
    *   En tant que utilisateur, je veux pouvoir supprimer une conversation, afin de retirer définitiveement les discussions non pertinentes de ma vue.

### Fonctionnalités de recherche et de statut

*   **US-MSG-009 : Rechercher dans les conversations**
    *   En tant que utilisateur, je veux pouvoir rechercher un mot-clé dans mes conversations, afin de retrouver facilement une information.

*   **US-MSG-010 : Statut en ligne**
    *   En tant que utilisateur, je veux pouvoir voir si un autre utilisateur est en ligne, afin de savoir si je peux attendre une réponse rapide.
